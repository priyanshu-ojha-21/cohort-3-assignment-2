## 🛠️ Assignments Included

### 1. Astro Winter Armor II (Product Card)
* **Description:** A premium e-commerce product layout highlighting an advanced winter puffer jacket. 
* **Key Features:** Clean alignment, custom-built circular size selectors, interactive product variation grid, and drop-shadow depth effects.
* **Core Tech:** Flexbox layouts, specific element-level margins, image filtering/shadowing.

### 2. ScaleX (Insights & Blogs)
* **Description:** A modern, dark-themed blog and insights section matching production-ready SaaS platforms.
* **Key Features:** Active category filter buttons, multi-column card layout, smooth hover lift effects, and custom text letter-spacing.
* **Core Tech:** CSS Grid, transparent color overlays (`rgba`), flex-wrap adjustments.

### 3. Reload Designs (Visual Culture Landing Page)
* **Description:** An agency-style, typography-heavy aesthetic layout emphasizing clean structure and editorial visuals.
* **Key Features:** Side-aligned section titles, unique inline heading images, and robust bold headers.
* **Core Tech:** Combined Flexbox layouts, uppercase tracking, editorial grid-like alignments.

---

## 🧠 Key Technical Takeaways
* **Semantic HTML5:** Built clean structures from scratch using header, section, footer, and semantic navigation elements.
* **Advanced Responsiveness:** Implemented custom media queries for different breakpoints (tablets and mobile phones) to switch gracefully from grid rows to single-column stacking.
* **Human-Written Code:** Prioritized clean, readable class naming conventions and modular properties over bloated frameworks.
